Conexión en vivo — API BDEstat del Banco Central de Chile

El Banco Central ofrece acceso gratuito a su base estadística (BDEstat). Primero regístrate en si3.bcentral.cl (es gratis, solo necesitas un correo). Luego ingresa tus credenciales para intentar cargar datos en tiempo real de tipo de cambio y TPM directamente desde tu navegador.

⚠️ En esta versión de archivo compartido, la conexión se hace directo desde tu navegador (sin el puente de Cowork). Si el Banco Central bloquea la solicitud por política de origen cruzado (CORS), verás un error abajo — es una limitación del navegador, no de tus credenciales. El resto del dashboard funciona igual, con datos ya incluidos en el archivo.

📚 Fuentes de datos — Cómo acceder a cada una

🏦 Banco Central de Chile

API REST gratuita (requiere registro) para tipo de cambio, tasas, TPM, IPC, IMACEC. PIB Regional (trimestral, sin registro) con desagregación por las 16 regiones. Datos diarios y mensuales desde los años 80.

→ si3.bcentral.cl (registro gratis) → PIB Regional (público, sin registro)

📊 INE — Inst. Nacional de Estadísticas

ENE nacional y regional (empleo/desempleo, desagregado por sexo), IPC, Censo 2017, proyecciones de población. Se actualiza el mismo día de la publicación oficial (calendario INE).

→ ine.gob.cl → Banco de Datos ENE (consulta en vivo)

💼 SII — Impuestos Internos

Empresas activas, renta, IVA, declaraciones y estadísticas tributarias por región y actividad económica. Datos anuales.

→ sii.cl/estadisticas

🌐 Banco Mundial

API completamente abierta y sin registro. PIB, inflación, población, pobreza. Los indicadores nacionales anuales de este monitor (PIB, PIB per cápita) provienen de aquí; el desempleo usa el dato oficial INE más reciente.

→ data.worldbank.org

📈 PIB per cápita, Población y PIB Regional

PIB per cápita y población (Base 2024, Censo 2024) confirmados en la tabla "Indicadores macroeconómicos anuales" de la Base de Datos Estadísticos del Banco Central, de acceso público. PIB Regional trimestral también público, sin registro.

→ BDE: Indicadores macroeconómicos anuales → Anuario Cuentas Nacionales 2018-2025

🏘️ CASEN — Min. Desarrollo Social

Encuesta de Caracterización Socioeconómica Nacional. Pobreza, ingresos y acceso a servicios por región. Se levanta cada 2–3 años.

→ observatorio.mds.cl

📁 datos.gob.cl

Portal oficial de datos abiertos. Centraliza datasets de INE, SII, Ministerios y otros organismos en CSV, JSON y XLS.

→ datos.gob.cl

ℹ️ Sobre este archivo

Esta es una versión estática exportada el 6 de julio de 2026 del Monitor Económico y Social de Chile, pensada para compartir por correo o carpeta compartida. Todos los datos (excepto la conexión opcional al Banco Central en la pestaña correspondiente) están incrustados en este mismo archivo — no requiere internet para verse, salvo para cargar los gráficos (vía CDN) y las conexiones opcionales.

Es un archivo de un solo momento en el tiempo: no se actualiza solo. Para ver la versión con actualización automática semanal, contacta a quien administra el Monitor Chile en Cowork.
